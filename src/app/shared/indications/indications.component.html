/ A LA HORA DE REGISTRARTE

Recordar siempre tu PSEUDÓNIMO y tu CÓDIGO DE MATRICULA, ya que será necesario si quieres volver a iniciar sesión en el sistema y seguir con el test.

Algunas Recomendaciones :
# El Pseudonimo debe contener una palabra, de esta manera te recordarás más rapido.

/ ¿QUIERES VOLVER A INICIAR SESIÓN NUEVAMENTE?

/ PERMITIR MICRÓFONO ( UNA VEZ INICIADO SESIÓN)

Tenemos que dar acceso a nuestro micrófono, de otra manera no se podrá avanzar en el test.

Algunas Recomendaciones :
# En la sección TEXTO para comenzar con el reconocimiento de voz, tienes que presionar el siguiente icono. # Ser lo más claros a la hora de pronunciar las palabras.

/ COMO FUNCIONA EL SISTEMA DE NIVELES ( UNA VEZ INICIADO SESIÓN)

Se tiene 3 niveles, de los cuales uno trás otro se van desbloqueando.
Si hemos llegado al final (nivel 3), entonces todo el puntaje acumulado en los tres niveles se mostrarán en la parte izquierda superior.

Algunas Recomendaciones :
# El Test no acaba con solo terminar el tercer nivel. Sino puedes seguir intentando las veces que quieras y seguir acumulando puntajes.
